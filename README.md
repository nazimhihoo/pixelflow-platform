# PixelFlow

Official PixelFlow by NX - AI image generation platform for web and Android with backend proxy, image history, and gallery support.

## Vision
PixelFlow is an AI image generation platform designed for web and mobile use. The goal is to provide a smooth prompt-to-image experience with saved history, gallery access, and scalable backend integration.

## Planned Features
- AI image generation
- External API integration through secure backend proxy
- Prompt history
- User gallery
- Download and share support
- Web app + Android app support
- Firebase integration
- GitHub-based development workflow

Set up the base repository, project structure, documentation, and development workflow.
